# Installing PbTe_Thermoelectric_PropertiesApp

## System Requirements
- Windows 10 or 11 (64-bit)
- MATLAB Runtime--MATLAB version (R2024b)
  - Download from: https://www.mathworks.com/products/compiler/matlab-runtime.html

## Steps to Install and Run

**Install MATLAB Runtime**
   - Go to the link above and download the installer for your system.
   - Run the installer and follow the on-screen instructions.
